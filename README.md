Expense Tracker

It contains 3 modules
1, Add Expense
2. View Expenses
3. Delete Expense

Tech Stack:
1. Python
   - FASTAPI
   - MySQL Connector
2. MYSQL Database
3. HTML
4. Javascript


Command to API to run
- go to app folder
> uvicorn app:app --reload --port 8000

Command to run http server
- go to ui folder
> python -m http.server 8088
